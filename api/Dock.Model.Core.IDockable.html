DockGroup

Gets or sets the docking group identifier. Dockables can only be docked with other dockables that have the same group identifier. If null or empty, the dockable can be docked with any other dockable. This enables creating isolated docking areas where only specific types of content can be docked together.

string? DockGroup { get; set; }
