Namespace

Indicates that an interface requires a DataTemplate in DockControl. This attribute is used to mark dock-related interfaces that need corresponding DataTemplates for proper rendering in the dock system.

[AttributeUsage(AttributeTargets.Interface, AllowMultiple = false, Inherited = false)]
